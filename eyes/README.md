# README - `eyes`

## `ontology`
- ontology file (+path)

## `plot_type`
- choose `web` or `tree`

## `*_colour`
- colours for `superclass` (default orange) and `subclass` (default skyblue)

## `to_annotate_subclasses`
- choose `True` or `False` to overlay text for subclasses in plots
- recommended to use `False` if ontologies are large
